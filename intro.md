# Aerospace Design

Welcome to the Aerospace Design notes. This resource covers the fundamentals of fixed-wing aircraft design — from the philosophy and process of design, through the core technical disciplines, to the trade-off methods used in real-world practice.

The notes are structured around ten sections, each building on the last. The approach mirrors that of a practising engineer: start broad, make educated assumptions, and refine progressively through iteration.

## How to use these notes

Each section is a self-contained Jupyter notebook. The markdown cells contain the theory and discussion; the code cells contain Python implementations of key relationships, and produce figures that are discussed in the text.

```{note}
These notes draw primarily from Raymer {cite}`raymer2018`, Torenbeek {cite}`torenbeek2013`, Anderson {cite}`anderson2012`, and Nicolai {cite}`nicolai2002`. Full references are provided at the end of each section.
```
